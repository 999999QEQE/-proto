# 本地桌面轮盘/随机器

这是一个可离线运行、容易编辑的桌面小程序，每个“页面”都是一组可切换的轮盘内容。每页都可设置标题、副标题、整屏背景图片/视频路径，以及用长篇段落组成的“轮盘”条目，并提供随机高亮动画和随机器。

## 快速开始
1. 确保已安装 Python 3（内置 Tkinter）。如需背景图缩放，建议安装 Pillow：
   ```bash
   pip install pillow
   ```
2. 在项目根目录运行：
   ```bash
   python main.py
   ```
   程序窗口将直接打开，无需浏览器或本地服务。

## 功能
- **多页面**：左侧列表可添加/删除页面，支持独立的标题、副标题和媒体路径。
- **整屏背景**：填写图片或视频文件路径，图片会铺满预览区；视频可点击“打开视频”用系统播放器全屏观看。
- **长段落轮盘**：在文本框内用空行分隔多个段落，点击“随机高亮段落”即可在段落间做动画并给出结果。
- **随机器**：自定义最小/最大值，生成随机数。
- **持久化**：所有页面配置保存在 `pages.json`，下次启动自动加载。

## 使用小贴士
- 图片预览依赖 Pillow；如果未安装，将在画布上提示“媒体预览不可用”。
- 视频文件不会在窗口内播放，但会调用系统默认播放器全屏展示，方便覆盖整个界面。
- 通过“保存当前页面”按钮可随时写入 `pages.json`；切换页面时也会自动保存当前编辑内容。

## 目录结构
- `main.py`：桌面应用入口与全部逻辑。
- `pages.json`：自动生成的配置文件，存储全部页面数据。
# 多页面轮盘编辑器

本项目提供一个本地即可运行、可随时编辑的轮盘小程序。每个轮盘占据独立页面，页面可自由新增，并支持标题、副标题、媒体（图片/视频）、轮盘条目以及随机器。

## 使用方式
1. 直接双击或拖拽 `index.html` 至浏览器即可使用；或在项目根目录运行本地服务：
   ```bash
   python -m http.server 8000
   ```
   然后访问 <http://localhost:8000>。
2. 左侧点击“添加新页面”即可创建新轮盘页面，右侧可填写标题/副标题、上传或粘贴媒体链接、录入轮盘条目并进行旋转抽取。
3. “随机器”可在自定义范围内生成随机数，所有数据保存于浏览器 `localStorage`，刷新后仍可继续编辑。

## 主要特性
- **多页面轮盘**：每个页面独立保存内容，侧栏可快速切换。
- **媒体支持**：图片/视频均可上传或填入链接进行展示。
- **可视化轮盘**：Canvas 绘制彩色扇区，带旋转动画与结果提示。
- **随机器**：自定义数值范围，快速生成随机数。
- **本地持久化**：所有配置自动保存到浏览器本地，无需额外后台。

## 自定义与编辑
- 轮盘条目：输入文本后点击“添加项”，条目可随时删除。
- 媒体：上传本地文件或填入网络地址。常见视频后缀（mp4/webm/mov）将以视频方式呈现，其余按图片显示。
- 布局：样式集中在 `style.css`，逻辑集中在 `app.js`，便于二次开发。
